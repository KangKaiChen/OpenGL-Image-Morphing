# OpenGL-Image-Morphing



2.(作業環境)
	作業環境:win11 64bit
	編譯器:Visual Studio 2022
  
3.(作業指定功能)
	已完成建立一個GUI介面，能夠顯示Load進來的圖片、變形後的結果以及可供操作的按鈕
	已完成按鈕功能load Input Image、Add/Copy/Move/Remove Point、Image Morphing、Save Output Image、Quit、Reset
	已完成Slider bar:越往左越接近圖片1，越往右越接近圖片2，且旁邊需顯示目前的數值(0為圖片1，m為圖片2，m取決於輸出結果圖片的張數)

  
4.(敘述如何操作我的作業)
	load image: 可以選擇要輸出的圖片
	load point:	可以讀取點
	add point:	可以增加點
	copy point:	可以講img1上的點複製到img2上
	move point:	可以移動img1&img2上的點
	remove point:可以刪除img1&img2上的點
	save point: 可以儲存點
	morphing:	可以將img1&img2進行image morphing
	reset:	可以將所有動作重置
	quit:		可以退出程式
	slider bar:	滑動slider bar將morphing分成7部份去做變換
